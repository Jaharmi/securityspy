# SecuritySpy for Home Assistant

This is a Home Assistant Integration for [Ben Softwares](https://www.bensoftware.com) SecuritySpy Surveillance system.

Basically what this does, is integrating the Camera feeds from SecuritySpy in to Home Assistant, adds switches to adjust recording settings, adds Binary Motion Sensors and Sensors that show the current Recording Settings pr. camera.

** This program is under development and no working code is currently available **
